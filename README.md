# Pedalion Trees

Work in progress

## Try it Out

[https://perseids-publications.github.io/pedalion-trees/](https://perseids-publications.github.io/pedalion-trees/)

## Installation

`yarn install`

## Running the development server

`yarn start`

## Building for deployment

`yarn build`

## Deploying a new version to github.io

`yarn deploy`
